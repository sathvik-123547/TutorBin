# Screencast Shot List

Target length: 4 to 6 minutes.

## Shot 1 - Intro and objective (10-15s)

- Show app window.
- State this is a two-deck DJ app with interactive looper and custom Scratch/Vinyl mode.

## Shot 2 - Deck controls + widgets (30-45s)

- Focus on Deck A.
- Show at least five widgets (PLAY, STOP, LOAD, gain, speed, position, scratch controls).
- Start and stop playback.
- Continuously adjust gain.

## Shot 3 - File loading with drag/drop (20-30s)

- Drag audio file from file explorer onto Deck A.
- Repeat for Deck B.

## Shot 4 - Waveform, timecode, playhead (25-35s)

- Start playback and zoom attention to waveform.
- Show timecode updating and playhead moving.

## Shot 5 - Two decks and crossfader mix (35-50s)

- Play both decks simultaneously.
- Move crossfader from left to right and back.
- Make the audible change obvious.

## Shot 6 - Looper visualization and interaction (45-70s)

- Click-drag on waveform to create loop region.
- Show red translucent loop overlay.
- Let playback run long enough to hear repeated loop.
- Adjust loop while playing.

## Shot 7 - Custom feature demo: Scratch/Vinyl mode (60-90s)

- Toggle SCRATCH ON on one deck.
- Right-click drag waveform to scratch.
- Demonstrate sensitivity slider impact.
- Demonstrate inertia slider impact (release and let speed decay naturally).

## Shot 8 - Wrap-up (10-20s)

- Recap that all coursework requirements are demonstrated.
- Mention custom feature explanation text file is included.
